# 🎼 My Music Companion
### CCBC Praiseland

A mobile-ready music companion app for worship teams.

---

## 🚀 Deploy to Vercel (Step by Step)

### Option A: Drag & Drop (Pinaka-madali!)

1. Install Node.js from https://nodejs.org
2. Open Terminal / Command Prompt in this folder
3. Run: `npm install`
4. Run: `npm run build`
5. A `dist/` folder will appear
6. Go to https://vercel.com → Sign up free
7. On the dashboard, **drag and drop** the `dist/` folder
8. Done! You'll get a link like `my-music-companion.vercel.app` 🎉

### Option B: Via GitHub (para ma-update anytime)

1. Go to https://github.com → Create account
2. Create new repository → Upload this entire folder
3. Go to https://vercel.com → "Add New Project"
4. Connect your GitHub → Select the repo
5. Click Deploy!

---

## 📱 Install on Phone (After deploying)

**Android:**
- Open the link in Chrome
- Tap the 3 dots menu → "Add to Home Screen"

**iPhone:**
- Open the link in Safari
- Tap Share button → "Add to Home Screen"
